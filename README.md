# 🏠 Hostel Maintenance System

A web-based application developed using Flask to manage hostel maintenance complaints efficiently. Students can raise complaints, and admins can track, assign workers, and update status in real-time.

---

## 🚀 Features

- 👩 Student Login (PRN based)
- 👨 Admin Login (Email based)
- 📝 File Complaints (Electrician, Plumbing, Mess, etc.)
- 📊 Complaint Status Tracking
- 🔄 Admin Panel for Updating Status & Assigning Workers
- 📅 Date & Time Tracking for Complaints
- 📈 Dashboard with Pie Chart (Chart.js)
- 🎨 Interactive UI with animations and colors

---

## 🛠️ Technologies Used

- Python (Flask)
- SQLite Database
- HTML, CSS
- JavaScript (Chart.js)
